Add in a sidebar widget to display a book of the month.

== Description ==
Add a sidebar widget to display information about a book using Google's Book API to display information about the book.

== Installation ==
1. Upload everything inside .zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
